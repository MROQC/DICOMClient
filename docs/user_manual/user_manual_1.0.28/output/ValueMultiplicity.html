    DICOM Value Multiplicity Table

    Name Description
    0Does not contain values (such as a Sequence)
    1Must have exactly one value
    1-2Can have one or two values
    1-3Can have from one to three values
    1-32Can have from one to 32 values
    16Must have exactly 16 values
    1-8Can have from one to 8 values
    1-99Can have from one to 99 values
    1-NCan have from one to any number of values
    2Must have exactly 2 values
    2-2NCan have from 2 to any even number of values
    2-NCan have from 2 to any number of values
    3Must have exactly 3 values (often used for 3D coordinates)
    3-3NMust have a at least 3 values up to any multiple of 3 values
    3-NCan have from 3 to any number of values
    4Must have exactly 4 values
    6Must have exactly 6 values
    6-NCan have from 6 to any number of values
    9Must have exactly 9 values
