    Anonymizing Files

    Before anonymizing, first make sure that the application is in anonymize mode using the Anonymize   Upload radio buttons at the top of the main window.


    If a particular patient ID is desired, enter it in the New Patient ID field. This field is initialized to a random value at startup. The pattern for this is defined in the configuration file.


    Be default, the patient name is set to the same value as the patient ID. If they need to be different, click the checkbox to the right of the New Patient Name field, which will enable the field and a different name to be entered.


    To anonymize a series, click the Anonymize button to the right of the series. When complete, the white on green plus sign will indicate that the series has been anonymized.


    One of the features of the program is to properly handle the anonymization of UIDs across files within a series and study. (more info) After anonymization, the generated files will have new UIDs and the relationships between files will be maintained, provided that the files were anonymized in the same session (program invocation).

    When files are anonymized, they are put in the Destination shown near the top of the main screen. The destination may be changed by:

    • Using the Browse button
    • Copy and pasting a folder name into the field
    • Typing into the field
    If the Destination does not exist, it will be created before anonymization begins. If you do not have permission to write files, then anonymization will fail. Note that repeatedly anonymizations will create a new set of files on each invocation.

    To anonymize an entire patient may be anonymized by clicking the Anonymize Patient button in the uppor right hand corner of each patient's information.

    Multiple patients may be anonymized by clicking the Anonymize All button at the bottom of the main window.
