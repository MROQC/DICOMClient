    Previewing Files

    To preview a DICOM series, click the Preview button to the right of the series. Previewing a file will not change it.

    All DICOM files may be viewed in text mode, and those with images may be viewed in image mode. Below is an annotated screen shot of the image preview:

    In text mode, the meta-data of the files is shown. Multi-valued attributes have their list of values display with a separating '\' between each value. DICOM sequences are shown with indenting to reflect the hierarchy.

    Optionally, the Details checkbox may be checked, which prefixes each attribute with the

    • group number
    • element number
    • value representation
    • value multiplicity

    The group and element numbers uniquely define the attribute. The value representation indicates the type of value (string, floating point, etc), and the value multiplicity indicates how many values may be specified. For example, a value multiplicity of 1 indicates that the attribute must have exactly one value, and a value multiplicity of 1-N indicates that the attribute must have one or more values. See Value Representations and Value Multiplicity for details.
